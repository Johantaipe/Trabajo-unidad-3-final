Configuración de Google OAuth2 (Admin)
Para activar el login con Google real:
  1. Cree credenciales en la Consola de Google Cloud.
  2. Origen de JS autorizado: http://localhost:8080
  3. URI de Redirección autorizada: http://localhost:8080/auth/google-login
  4. Copie el Client ID generado y reemplace el valor en login-cliente.html (línea 215).
  5. Para producción, defina las variables de entorno de su servidor de despliegue (ej. Render/Heroku): GOOGLE_CLIENT_ID y GOOGLE_CLIENT_SECRET, o en su archivo application-prod.properties.
