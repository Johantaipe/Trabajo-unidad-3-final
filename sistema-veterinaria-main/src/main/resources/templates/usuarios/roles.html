Roles y Niveles de Acceso

Lista de Usuarios Nuevo Usuario Roles de Acceso

👑 ADMINISTRADOR

Único usuario con control total sobre la configuración del sistema y cuentas de acceso.

Permisos Clave:
  • Gestión de Usuarios
  • Restablecer contraseñas
  • Configuración general
  • Acceso total a módulos

💼 SECRETARIA

Gestión de la recepción, agenda, control de caja y registro básico de clientes/mascotas.

Permisos Clave:
  • Control de Agenda y Citas
  • Caja y Registro de Ventas
  • Clientes / Propietarios
  • Inventario (solo consulta)
  • Bloqueado: Crear usuarios
  • Bloqueado: Configuración

🩺 VETERINARIO

Acceso exclusivo a módulos médicos, clínicos y hospitalización de pacientes.

Permisos Clave:
  • Historias Clínicas
  • Vacunación / Fórmulas
  • Hospitalizaciones
  • Consultas médicas
  • Bloqueado: Caja y Ventas
  • Bloqueado: Usuarios

🐾 CLIENTE

Portal externo para dueños de mascotas. Sin acceso al panel interno de control.

Permisos Clave:
  • Portal de Cliente Web
  • Ver citas y vacunas de su mascota
  • Ver comprobantes de pago
  • Bloqueado: Sistema interno
